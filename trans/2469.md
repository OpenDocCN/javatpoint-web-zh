在本文中，我们将详细了解 bootstrap4 头。

### Bootstrap 4 Header 是什么意思？

在 Bootstrap 4 中，标题是位于页面顶部的导航栏。导航栏可以扩展或折叠，具体取决于屏幕大小。标准导航栏是在的帮助下创建的

```

<nav class="navbar navbar-default">

```

**我们来看看 Bootstrap 4 Header 的各种例子。**
