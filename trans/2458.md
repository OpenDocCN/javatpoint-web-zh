# 自举 4 吐司

> 原文:[https://www.javatpoint.com/bootstrap-4-toast](https://www.javatpoint.com/bootstrap-4-toast)

在本文中，我们将了解 bootstrap 4 toast 示例。

### 自举 4 吐司是什么意思？

Bootstrap 4 中的 toast 元素类似于一个警告框。这是一个轻量级通知，旨在模仿移动和桌面操作系统已经普及的推送通知。Bootstrap 4 toast 是在 flexbox 的帮助下创建的，因此它们很容易对齐和定位。

**我们来举一下 Bootstrap4 吐司的各种例子。**

### 例 1:

```

<! DOCTYPE html>
<html lang="en">
<head>
  <title> Bootstrap4 Toast Example </title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">
  <link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/css/bootstrap.min.css">
  <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.5.1/jquery.min.js"> </script>
  <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.16.0/umd/popper.min.js"> </script>
  <script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.5.2/js/bootstrap.min.js"> </script>
</head>
<style>
body {
  background: #00BCD4;
  padding-top: 100px;
  .container {
    background: #fff;
    padding: 50px 100px;
    max-width: 700px;
    text-align: center;
    @media(max-width: 699px) {
      width: 90%;
      padding: 30px 10px;
    }
    p {
      font-size: 18px;
    }
    .button {
      text-align: center;
    }
  }
}
h3 {
  font-family: sans-serif;
  display: block;
  font-size: 2rem;
  font-weight: bold;
  text-align: center;
  letter-spacing: 3px;
  color: hotpink;
  text-transform: uppercase;
  margin-top: 20px;
}
button:hover {  
  color: #1A33FF;  
}
</style>
<body>
<div class="container">
  <h3> Boostrap4 | Toast Example </h3>
  <div class="toast" data-autohide="false">
    <div class="toast-header">
      <strong class="mr-auto text-primary"> RAM </strong>
      <small class="text-muted"> 15 mins ago </small>
      <button type="button" class="ml-2 mb-1 close" data-dismiss="toast">  </button>
    </div>
    <div class="toast-body">
      Hello, how are you!
    </div>
  </div>
</div>
<script>
$(document).ready(function() {
  $('.toast').toast('show');
});
</script>
</body>
</html>

```

**说明:**

在上面的示例中，我们创建了一个 Bootstrap 4 toast 示例。在本例中，我们使用了 data-autohide="false "来默认显示吐司。
