# 聚合物铂元素

> 原文：<https://www.javatpoint.com/polymer-platinum-elements>

聚合物铂金元素用于添加推送通知、离线使用、与附近的蓝牙设备通信等功能。并提供将网页转换为真正的 web 应用程序的功能。

参见聚合物中的铂元素。
