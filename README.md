## 下载

### Docker

```
docker pull apachecn0/javatpoint-web-zh
docker run -tid -p <port>:80 apachecn0/javatpoint-web-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g javatpoint-web-zh
javatpoint-web-zh <port>
# 访问 http://localhost:{port} 查看文档
```
