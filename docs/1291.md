# 主干. js 集合

> 原文:[https://www.javatpoint.com/backbonejs-collection](https://www.javatpoint.com/backbonejs-collection)

集合是一组有序的模型。

它用于处理一组相关的模型。

它处理新模型到服务器的加载和保存。

它提供了帮助函数来对模型列表执行聚合和计算。

您可以通过扩展主干的集合类来创建自己的集合。

如果在集合中的模型上触发了事件，那么这也将直接在集合上触发。它便于您监听集合中任何模型中特定属性的更改。
