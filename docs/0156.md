# HTML 标签

> 原文:[https://www.javatpoint.com/html-thead-tag](https://www.javatpoint.com/html-thead-tag)

HTML<和>元素用于定义 HTML 表格的标题。<和>标签与<标签一起使用，后者定义了 HTML 表格中的表格标题、表格正文和表格页脚。

<标签和>标签必须是<表>元素的子元素，并且必须在任何< tbody >、< tr >或< tfoot >元素之前使用。

<标签和>标签至少要包含一行< tr >元素。

### 语法

```

    <thead>
  		<tr>
  		</tr>
  	</thead>

```
