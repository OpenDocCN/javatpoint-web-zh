# 物化 CSS 转盘

> 原文:[https://www.javatpoint.com/materialize-css-carousel](https://www.javatpoint.com/materialize-css-carousel)

物化 CSS 轮播提供了一个健壮和通用的方式来制作图像滑块。这也是触摸使能，当我们在移动设备上使用它时，它给人一种流畅的体验。

**JavaScript**

```
<script>$(document).ready(function(){
      $('.carousel').carousel();
    });</script>

```

* * *
