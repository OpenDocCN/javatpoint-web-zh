# 根指令

> 原文:[https://www.javatpoint.com/sass-at-root-directive](https://www.javatpoint.com/sass-at-root-directive)

Sass @at-root 指令是嵌套规则的集合，用于设置文档根的样式块。

**语法:**

```

@at-root (without: ...) and @at-root (with: ...) 

```
