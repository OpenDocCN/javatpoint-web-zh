## remove()方法

JavaScript 中使用 remove()方法从选择列表中移除选定的元素。

### 语法 1:

```

$(selector).remove (selector)

```

**例:**$(“p”)。移除()；

### 语法 2:

```

$("#FIELDID option[value='X']").remove();

```

**示例:**$(“# select 1 option[value = ' hello ']”)。移除()；
