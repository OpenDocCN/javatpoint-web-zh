# CSS 箭号

> 原文：<https://www.javatpoint.com/css-arrow>

CSS 箭头用于添加数组和工具提示。它使工具提示像一个语音气泡。它也可以用四种方式表示:

*   顶部箭头
*   底部箭头
*   向左箭头
*   向右箭头

* * *

## CSS 顶部箭头

当您将鼠标光标移动到元素上时，顶部箭头用于在工具提示的顶部添加类似箭头的结构。它将在元素的底部显示工具提示。

**见本例:**

```css

顶部箭头示例
将鼠标光标移到下面的标题上
**Welcom to JavaTpoint**
A solution of all technology

```
