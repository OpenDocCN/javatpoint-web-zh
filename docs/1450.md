## 标准 CSS

纯粹的。CSS 非常容易学习，因为它使用标准的 CSS。它不依赖于任何外部 JavaScript 库，比如 jQuery。

## 最小尺寸

纯粹的。CSS 的设计非常简约和扁平。它非常小，整个模块集可以在 4.0 KB 内缩小和压缩。它的设计方式是，用户可以轻松地添加新的 CSS 规则，而不是覆盖现有的 CSS 规则。
