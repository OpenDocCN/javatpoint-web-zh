# 框架 7 浮动操作按钮

> 原文:[https://www.javatpoint.com/framework7-action-button](https://www.javatpoint.com/framework7-action-button)

框架 7 浮动操作按钮用于提升的操作。它就像一个在用户界面上方浮动的圆圈图标，具有包括变形、启动和转移锚点在内的运动行为。

#### 注意:浮动动作按钮仅在材质主题中受支持。

框架 7 中使用了 3 种类型的操作按钮:

1.  **浮动动作按钮布局:**很简单。您只需将其作为页面或视图的子级。
2.  **变形为 popover:** 如果你想在点击浮动动作按钮时打开 Popover，那么你可以使用浮动按钮到 Popover 类。
3.  **快速拨号:**使用快速拨号点击浮动动作按钮，即可调用相关动作。

## 浮动操作按钮布局

这是一个非常简单的布局。您只需将其作为页面或视图的子级。使用？浮动按钮？类在应用程序中使用浮动操作按钮。
