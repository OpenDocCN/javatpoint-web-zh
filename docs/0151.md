# HTML 标签

> 原文：<https://www.javatpoint.com/html-td-tag>

HTML<TD>标签用于指定 HTML 表格中包含表格数据的单元格。< td >标签必须是< tr >(表格行)标签的子元素。每个表格行可以包含多个< td >数据元素。

一个< tr >标签的分组< td >元素在表格中呈现为一行。默认情况下，< td >元素的内容是规则的，并且在表格中左对齐。

### 语法

```

  <td>.......</td>  

```
