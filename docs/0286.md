# 内部 CSS

> 原文：<https://www.javatpoint.com/internal-css>

内部样式表用于为单个文档添加独特的样式。它在

示例:

```css
<!DOCTYPE html>
<html>
<head>
<style>
body {
    background-color: linen;
}
h1 {
    color: red;
    margin-left: 80px;
} 
</style>
</head>
<body>
<h1>The internal style sheet is applied on this heading.</h1>
<p>This paragraph will not be affected.</p>
</body>
</html>

```
