# Angular镜

> 原文：<https://www.javatpoint.com/angularjs-scopes>

范围是一个对象，它被指定为 HTML(视图)和 JavaScript(控制器)之间的绑定部分。它起到连接控制器和视图的作用。它可用于视图和控制器。

* * *

## 如何使用范围

要在 AngularJS 中创建控制器，必须将$scope 对象作为参数传递。

**见本例:**

```

# { {肉名}}

属性“carname”是在控制器中创建的，可以通过使用{{ }}括号在视图中引用。

```
