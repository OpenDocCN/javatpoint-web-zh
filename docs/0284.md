# 如何添加 CSS

> 原文:[https://www.javatpoint.com/how-to-add-css](https://www.javatpoint.com/how-to-add-css)

CSS 被添加到 HTML 页面中，以根据样式表中的信息格式化文档。在 HTML 文档中插入 CSS 有三种方法。

1.  内嵌 CSS
2.  内部 CSS
3.  外部 CSS

* * *

## 1)内嵌 CSS

内联 CSS 用于在单行或单个元素上应用 CSS。

例如:

```

```
