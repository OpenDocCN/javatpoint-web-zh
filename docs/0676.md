## 什么是 jQuery

*   jQuery 是一个小型轻量级的 JavaScript 库。
*   jQuery 是跨平台的。
*   jQuery 的意思是“少写多做”。
*   jQuery 简化了 AJAX 调用和 DOM 操作。
