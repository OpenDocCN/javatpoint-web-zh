```js
ngAnimate 是做什么的？
ngAnimate 模块本身不动画化 HTML 元素。它只添加和删除一些预定义的类，以便在 ngAnimate 注意到某些事件时制作动画，例如隐藏或显示一个 HTML 元素。
以下是 AngularJS 中添加/删除类的指令列表:

*   ng 显示
*   隐藏
*   ng 级
*   ng-view
*   ng-包括
*   ng 重复
*   条件判断指令
*   ng 开关

```
