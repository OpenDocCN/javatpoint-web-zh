# Angular 模块

> 原文：<https://www.javatpoint.com/angularjs-modules>

在 AngularJS 中，一个模块定义一个应用程序。它是应用程序不同部分的容器，如控制器、服务、过滤器、指令等。

模块用作 Main()方法。控制器始终属于一个模块。

* * *
