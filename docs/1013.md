# JSON 评论

> 原文:[https://www.javatpoint.com/json-comments](https://www.javatpoint.com/json-comments)

JSON 不支持评论。这不是一个标准。

但是您可以做一些技巧，例如在 JSON 对象中添加额外的注释属性，例如:

```

{
    "employee": {
        "name":       "Bob", 
        "salary":      56000, 
        "comments":    "He is a nice man"
    }
}

```

在这里，“注释”属性可以被视为注释。
