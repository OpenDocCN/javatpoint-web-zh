# CSS 页面设置

> 原文：<https://www.javatpoint.com/css-pagination>

CSS 分页是一种非常有用的技术，用于在主页上索引网站的不同页面。如果你的网站有很多页面，你必须给每个页面添加一些分页。

以下是不同类型的分页:

* * *

  *   [5](#)
