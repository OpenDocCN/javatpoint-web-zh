## 什么是 JSON

*   JSON 代表 JavaScript 对象符号。
*   JSON 是轻量级数据交换格式。
*   JSON 比 XML 更容易读写。
*   JSON 是独立于语言的。
*   JSON 支持数组、对象、字符串、数字和值。

## JSON 示例

在本教程中，您将获得大量 JSON 示例来很好地理解该主题。JSON 文件必须用保存。json 扩展。让我们看一个简单的 JSON 例子。
