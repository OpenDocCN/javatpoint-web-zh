> 哎哎哎:# t0]https://www . javatppoint . com/html-tr 标签

HTML<tr>标签用于定义表格中的行。< tr >标签可以由一个或多个<第>个标题单元格和< td >个数据单元格组成，以定义一行 HTML 表格。

<tr>标记必须是<表>元素的直接子元素，也可以是<和>、< tbody >和< tfoot >元素的嵌套子元素。

### 语法

```html

    <tr>.......</tr> 

```
