## 引导垂直滚动间谍菜单

您也可以使用垂直导航药丸作为菜单。

**见本例:**

```

  Bootstrap Example

<nav class="col-sm-3" id="myScrollspy">

*   [第一节](#section1)

*   [第二节](#section2)

*   [第三节](#section3)

*   [第四节](#) 

    *   [第 4-1 节](#section41)

    *   [第 4-2 节](#section42)

    </nav>
```
